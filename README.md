## 📋 What is this?

This project is an example of how to build a web app using Next.js with a Node.js backend. It shows the basics of putting together these two technologies so you can create fast, server-driven websites.

Next.js is a tool that helps make websites that load quickly and work well on many devices. Node.js runs the part of the app that handles data and logic behind the scenes.

This example helps you see how these parts connect, so you can start your own web app with this setup.

## 💻 How to Install and Run on Windows

Depending on which file you downloaded, follow the instructions below:

### If you downloaded a `.exe` file:

1. Locate the `.exe` file on your computer.

2. Double-click the file to start it.

3. If Windows asks, allow the app to run or install.

4. Follow on-screen prompts to finish installation if any appear.

5. Once done, the app may open automatically. Look for a new window with the app interface.

### If you downloaded a `.zip` file:

1. Find the `.zip` file you downloaded.

2. Right-click the file and choose “Extract All”.

3. Pick a folder to extract the contents to. Using the Desktop or Documents folder works well.

4. After extraction, open the new folder.

5. Look for a file named `start.bat` or `run.bat`. This file will launch the app.

6. Double-click the `.bat` file. A command window may pop up briefly to start the server.

7. Open your web browser.

8. Type `http://localhost:3000` into the browser’s address bar and press Enter.

The app should load, showing the example project running locally on your PC.
